Neste projeto entregue pelo nosso grupo, estamos testando e percebendo a diferença
entre fazer buscas inteligentes e outras não inteligentes assim em bancos de dados
muito grandes.

Nossa avaliação se resume a um código principal chamado "AVIIEstDados.py", uma biblioteca
onde está o código com a árvore AVL como uma classe, chamada "AVLTree.py", e uma base
de dados chamada "bd_avl.txt".

Em outros arquivos de texto apresentaremos com detalhes e clareza os resultados
obtidos nesse projeto, bem como sobre a base de dados utilizada.
